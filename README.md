---

## 🛠️ Getting Started

### 📦 Install dependencies
```bash
bun install
```

### ▶️ Run the app
```bash
bun backend/index.js
```

> 💡 Make sure you have **Bun v1.0.20+** installed:  
> 👉 [Install Bun →](https://bun.sh/docs/installation)
